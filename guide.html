Features
기능 안내
📊
대시보드
로그인 후 첫 화면. 올해 진행 캠페인 수, 이번 달 예정 건수, 실발송 합계를 확인하고 파이프라인 총 건수·예상 매출·확정 예정 현황을 한눈에 파악할 수 있습니다. 나의 캠페인과 전사 캠페인 단계별 현황도 함께 표시됩니다.
KPI 카드 단계별 현황 파이프라인 요약
📅
캘린더
발송 예약일 기준으로 캠페인을 월·주·일 뷰로 확인합니다. 카테고리·매체사·광고주·담당자 필터로 원하는 일정만 빠르게 조회할 수 있으며, 일정 클릭 시 캠페인 기본 정보가 팝업으로 표시됩니다.
월/주/일 뷰 다중 필터 미리보기 팝업
📋
캠페인 목록
등록된 캠페인 전체를 테이블로 조회합니다. 카테고리·상품·매체사·담당자·조회기간 등 다중 필터를 지원하며, 기본 조회기간은 현재 월입니다. 엑셀 다운로드와 신규 캠페인 등록 기능을 제공합니다.
다중 필터 엑셀 다운로드 캠페인 등록
🏗️
영업 리포트
칸반 탭: MAKEUP → POTENTIAL → COMMITMENT → ACTUAL 4단계로 딜을 관리합니다. ACTUAL 단계 딜은 [캠페인 전환] 버튼으로 캠페인 등록 폼이 자동 완성됩니다.
통계 탭: 본부·팀별 월별 목표·현황·달성률을 확인하고, 관리자는 목표 수치를 직접 수정할 수 있습니다.
칸반 보드 캠페인 전환 목표 관리 달성률 통계
📈
월별 발송량
매체사별·광고주별·대행사별·캠페인별 탭으로 발송 실적을 분석합니다. 연도·월·카테고리·매체사 필터를 조합해 원하는 기간의 총 매출액·매입액·매출이익을 집계할 수 있으며 엑셀 다운로드를 지원합니다.
4개 탭 분석 매출이익 집계 엑셀 다운로드
💰
정산
캠페인·매체·광고주·대행사별 뷰로 정산 현황을 관리합니다. 매출액·매입액·대행수수료·매출이익이 자동 계산되며, 입금·지급 상태 토글과 세금계산서 번호 입력을 지원합니다.
자동 금액 계산 입금·지급 관리 세금계산서
⚙️
관리 메뉴 (관리자 전용)
매출처 관리: 광고주·대행사 등록, 브랜드(하위 상품) 카테고리별 관리
매체 관리: 매체사 단가·수수료·담당자 정보 등록 및 수정
사용자 관리: 사용자 등록·소속·권한(영업/운영) 설정, 비밀번호 초기화

⚠️ 테스트 시 주의사항
• 비밀번호 변경 후 잠길 경우 관리자(admin)로 초기화할 수 있습니다.
• DB 연동 완료되어 데이터 생성, 수정 모두 가능합니다.
• 수정/삭제 버튼 조회되는 상황 : 본인이 작성자, 담당자인 경우 / admin 계정으로 로그인한 경우
• 사이트상에서 보이는 데이터는 실제 데이터를 참고하여 생성된 가상의 데이터입니다.
• '대시보드'의 '나의 캠페인' 블록의 데이터는 '로그인 한 사용자'가 '담당자'로 지정된 캠페인이 조회됩니다.
• '영업리포트' 메뉴는 브레인큐브_주간보고_2026년 시트를 참고하여 개발되었습니다. 상태가 active인 영업건의 경우 캠페인 생성이 가능합니다.
• '매출처 관리' 메뉴 생성하였으며, [브레인큐브 월마감정산서] 시트 참고하여 광고주/대행사 및 광고내용 리스트 생성했습니다. 캠페인 등록시 동작 로직 확인 가능합니다.
• 정산 메뉴도 추가되었습니다. 해당 메뉴에 대한 테스트 및 매체,매출처 등의 메뉴와 관련하여 수정/개선 필요한부분 확인 부탁드립니다.
• 정산메뉴 제외한 모든 메뉴는 모바일에서도 이용이 가능합니다.
