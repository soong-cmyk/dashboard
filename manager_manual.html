'성과입력완료만' vs '전체 캠페인'

정산 탭 상단 범위 필터는 집계 기준을 결정하는 가장 중요한 설정입니다.

성과입력완료만 (기본값)
실적이 확정된 캠페인만 집계합니다.

언제 사용:
확정된 매출·이익을 확인할 때. 정산 엑셀 다운로드 전 기준 설정으로 사용합니다.
전체 캠페인
진행 중인 캠페인까지 모두 포함합니다.

언제 사용:
이번 달 예상 매출 전체를 파악할 때. 아직 성과 미입력 캠페인도 포함되어 잠정 수치입니다.
다운로드 전 항상 범위 필터를 확인하세요. '전체 캠페인'으로 설정된 상태에서 다운로드하면 미확정 데이터가 포함됩니다.

이익률 계산 기준

대시보드와 정산 탭의 이익·이익률은 아래 산식으로 계산됩니다.

광고비 = 매출단가 × 정산수량 – 할인금액
매입액 = 매입단가 × 인정수량
매출수익 = 광고비 × 수수료율(%)
대행료 = 광고비 × 대행료율(%)
이익 = 매출수익 – 대행료
이익률 = 이익 ÷ 광고비 × 100 (%)
상품특이사항
SMS/LMS/MMS광고주 정산기준(실발송/예약)과 매체 정산기준이 별도로 설정되어 있어 정산수량과 인정수량이 다를 수 있습니다.
DA수량 개념 없이 광고비 총액 기준으로 정산합니다.
CPADB등록수(실적 건수)가 정산 기준입니다.
퍼미션콜이익 = 광고비 – OHC비용 – DNU비용. 수수료율·대행료율 미적용.

조직별 필터 활용

정산 탭 필터의 본부/팀을 선택하면 해당 조직 담당 캠페인만 집계됩니다.

대상설정 방법
전사 현황 (이사·대표)본부/팀 필터 선택 안 함 (전체)
특정 본부 (본부장)본부/팀 필터에서 해당 본부 선택
특정 팀본부/팀 필터에서 팀까지 세부 선택
연도·월 + 본부 필터를 조합하면 "1본부의 이번 달 정산 현황" 같은 형태로 빠르게 조회할 수 있습니다.
04

광고비 지급요청서

매체사에 광고비를 지급할 때 사용하는 엑셀 파일을 다운로드합니다.

다운로드 방법

  1. 1
    정산 탭에서 연도·월 필터를 지급 대상 월로 설정합니다.
  2. 2
    범위 필터를 '성과입력완료만'으로 확인합니다.
  3. 3
    우측 상단 ⬇ 다운로드 → 광고비 지급요청서를 클릭합니다.

파일 구성

매체사별로 별도 시트가 생성됩니다. 각 시트에는 해당 매체사의 캠페인 목록과 매입액·VAT·합계가 표시됩니다.

열내용
광고기간캠페인 발송일
캠페인명광고주_매체사 형식의 자동 생성 이름
광고비광고주에게 청구한 매출액 (참고용)
매입단가 · 매입액매체사 지급 기준 단가 및 금액
VAT매입액의 10%
합계매입액 + VAT (실제 지급 금액)
계산서세금계산서 발행 여부

퍼미션콜 시트

퍼미션콜 항목은 디앤유와 OHC 시트 2개로 분리 생성됩니다.

시트매입액 기준
디앤유동의건수 × 5,500원 (DNU 단가 고정)
OHC캠페인에 입력된 OHC 비용 합계
선입금 매체사는 지급요청서 시트에서 자동으로 제외됩니다.

화면 구성

각 세금계산서 요청 카드 상단에는 T-001 형식의 고유 번호가 표시됩니다. 실무자와 소통할 때 이 번호를 기준으로 확인하세요.

📤 매출 계산서 (세발)

광고주에게 발행하는 계산서. 실무자가 요청한 건이 미처리 상태로 표시됩니다.

📥 매입 계산서

매체사로부터 수취하는 계산서. 수취 완료 후 발행완료로 처리합니다.

발행 체크 방법

  1. 1
    세금계산서 탭에서 미처리 상태 항목을 확인합니다. 연·월 필터로 해당 월을 조회하세요.
  2. 2
    항목의 업체명, 공급가액, 품목을 검토하여 발행 준비가 되었는지 확인합니다.
  3. 3
    실제 세금계산서를 발행한 후, 해당 행의 상태를 발행완료로 변경합니다.
  4. 4
    매출 계산서의 경우 입금이 확인되면 입금완료로 업데이트합니다.

상태 기준

상태의미처리 주체
미처리실무자가 등록 요청했으나 아직 발행 전관리자 확인 필요
발행완료세금계산서 발행(또는 수취) 완료관리자 체크
입금완료매출 계산서 기준 실제 입금 확인됨관리자 체크
발행필요없음매입 계산서 중 계산서 수취가 불필요한 건관리자 체크

월 마감 전 미처리 건이 남아 있지 않은지 반드시 확인하세요.

매체사 관리

캠페인 등록 시 선택하는 매체 목록의 소스입니다. 좌측 메뉴 매체 관리에서 확인·등록·수정할 수 있습니다.

목록 화면

등록된 매체사 전체를 테이블로 표시합니다. 각 행을 클릭하면 상세보기로 이동하며, 좌측 토글로 활성/비활성을 전환할 수 있습니다. 비활성 매체사는 캠페인 등록 콤보박스에 표시되지 않습니다.

광고비 지급일자

매체에 광고비를 지급하는 기준일을 입력합니다 (예: 익월 15일, 익월 말일).

선입금으로 설정하면 정산 탭 → 지급요청서 엑셀 다운로드 시 해당 매체의 시트가 생성되지 않습니다.
선입금 매체는 별도 프로세스로 처리하므로 지급요청서 대상에서 자동 제외됩니다.

세금계산서 수취 여부

매체사로부터 세금계산서를 수취하는지 여부를 체크박스로 설정합니다.

설정값동작
체크 (기본값)세금계산서 탭에서 해당 매체 건이 정상 표시되고, 발행 체크 관리 대상에 포함
체크 해제세금계산서 탭에 해당 매체 건이 표시되지 않음. 세금계산서를 발행하지 않는 매체사에 설정
매체사 등록 후 캠페인을 등록해야 해당 매체가 정산·세금계산서 탭에 반영됩니다. 이미 등록된 캠페인은 설정 변경 즉시 적용됩니다.

브랜드 등록

각 매출처 아래에 브랜드를 여러 개 등록할 수 있습니다. 브랜드명과 카테고리(예: 금융, 통신)를 함께 입력합니다.

캠페인 등록 시 매출처를 선택하면 해당 매출처 소속 브랜드 목록이 자동으로 필터링됩니다. 브랜드가 없으면 직접 텍스트 입력도 가능합니다.

대행수수료율

대행사·랩사에 설정된 수수료율(%)은 캠페인 등록 시 정산 필드의 대행수수료에 자동으로 채워집니다. 캠페인별로 개별 조정이 가능합니다.
