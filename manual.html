01

캠페인 등록

새 캠페인을 등록하는 방법을 상품 유형별로 설명합니다.

등록 시작하기

  1. 1
    상단 오른쪽 + 캠페인 등록 버튼을 클릭합니다.
    캘린더에서 날짜 칸을 클릭해도 해당 날짜가 미리 입력된 상태로 등록 폼이 열립니다.
  2. 2
    등록 폼이 열리면 상품을 먼저 선택합니다. 상품에 따라 표시되는 필드가 달라집니다.
  3. 3
    필수 항목(*)을 모두 입력하고 등록 버튼을 클릭합니다.

02

캠페인 관리

등록된 캠페인을 검색하고 수정하는 방법을 설명합니다.

캠페인 찾기

좌측 메뉴 캠페인 목록에서 전체 캠페인을 조회합니다. 상단 필터로 범위를 좁힐 수 있습니다.

필터설명
검색창캠페인명(광고주_매체)이나 캠페인 ID로 검색합니다.
날짜 범위발송일 기준 시작일~종료일로 조회합니다.
상품SMS / DA / CPA 등 상품 유형별로 필터링합니다.
카테고리광고 카테고리별로 필터링합니다.
매체사 / 매출처 / 담당자특정 매체·광고주·담당자의 캠페인만 봅니다.
단계부킹확정 / 테스트완료 / 성과입력완료 등 진행 단계로 필터링합니다.

상세보기

목록에서 캠페인 행을 클릭하면 상세보기 화면이 열립니다.

📋 기본 정보

캠페인명, 발송일시, 상품, 매출처, 매체사, 담당자 등 등록 시 입력한 정보를 보여줍니다.

📊 성과 (2차 입력)

실발송수량, 클릭수, CTR 등 성과 데이터를 입력하고 확인합니다.

💰 수량 / 금액

정산수량, 매출단가, 매입단가, 광고비, 이익 등 금액 정보를 보여줍니다.

🎯 타겟 & 문구

타겟 조건, 디타겟 조건, 발송 문구(검수전/검수완료)를 관리합니다. DA·CPA는 발송문구 섹션이 표시되지 않습니다.

📝 특기사항

내부 메모, QA 체크, 발송 완료 확인 여부를 기록합니다.

📅 수정 이력

언제, 누가, 어떤 항목을 변경했는지 이력이 자동으로 기록됩니다.

캠페인 수정하기

  1. 1
    캠페인 상세보기 오른쪽 상단 수정 버튼을 클릭합니다.
  2. 2
    수정 화면에서 변경할 항목을 수정합니다. 등록 폼과 동일한 구조입니다.
  3. 3
    저장 버튼을 클릭합니다. 변경된 항목은 수정 이력에 자동 기록됩니다.
성과입력완료 단계 이후에도 수정이 가능하지만, 정산 관련 수치 변경은 신중하게 진행하세요.

복사 등록

기존 캠페인과 동일한 조건으로 새 캠페인을 빠르게 등록할 때 사용합니다.

  1. 1
    복사할 캠페인의 상세보기를 엽니다.
  2. 2
    오른쪽 상단 복사 등록 버튼을 클릭합니다.
  3. 3
    기존 캠페인의 정보가 등록 폼에 자동으로 채워집니다. 발송일시는 비어 있으므로 반드시 새로 입력합니다.
  4. 4
    내용을 확인·수정 후 등록 버튼을 클릭합니다.
복사 등록 시 발송문구, 타겟 조건도 함께 복사됩니다. 내용이 다르면 반드시 수정 후 등록하세요.

발송문구 입력 및 검수 처리

캠페인 상세보기의 타겟 & 문구 카드에서 발송문구를 관리합니다. (SMS/LMS/MMS/push/카톡msg 전용)

  1. 1
    타겟 & 문구 카드 오른쪽 ✏ 수정 버튼을 클릭합니다.
  2. 2
    발송문구 입력란에 최종 문구를 입력합니다.
  3. 3
    광고주 또는 매체사로부터 문구 검수가 완료되면 검수 완료 문구로 저장 체크박스를 활성화하고 저장합니다.

상세보기 문구 카드에는 검수전과 검수완료 탭이 있어 두 버전을 구분하여 확인할 수 있습니다.

검수완료 문구가 있으면 검수완료 탭이 우선 표시됩니다. 아직 검수 전이라면 검수전 탭을 확인하세요.

세금계산서 등록(발행 요청)

좌측 메뉴 세금계산서에서 매출·매입 세금계산서 발행 요청을 등록합니다. 등록 후 관리자가 발행 처리합니다.

세금계산서 등록은 발행 요청입니다. 실제 발행 완료 처리는 관리자가 확인 후 진행합니다.

불러오기

정산이 완료된 캠페인 데이터를 바탕으로 세금계산서 항목을 만드는 방법입니다.

  1. 1

  2. 2

  3. 3
    발행구분을 설정합니다. 항목별로 광고주 (매출 계산서) 또는 매체 (매입 계산서)를 선택합니다. 상단 일괄 설정 버튼으로 전체 일괄 변경도 가능합니다.
  4. 4
    다음(STEP 2)으로 이동하면 업체별 카드가 표시됩니다. 각 카드는 두 영역으로 구성됩니다.

    ① 수동 입력 항목 — 계산서에 실제로 기재할 품목을 직접 입력합니다.
    필드설명

    품목계산서 적요란에 기재할 내용. 예: 3월 LMS 광고비_SKT
    공급가액불러온 캠페인의 합산 금액이 자동 입력됩니다. 숫자 클릭 시 직접 수정할 수 있습니다.

  5. 5
    내용을 확인한 후 등록 버튼을 클릭합니다.

자주 묻는 질문

실무 중 자주 발생하는 질문과 답변입니다.

캠페인명을 직접 입력하고 싶은데 어디서 하나요?
캠페인명은 광고주 + 매체사 조합으로 자동 생성되며 직접 입력하는 칸이 아닙니다. 별도 프로모션 테마가 있다면 광고목적 필드에 입력하세요.
등록 후 발송일시를 바꾸고 싶어요.
상세보기 → 수정 버튼을 클릭한 뒤 발송일시를 변경하고 저장하면 됩니다.
실발송수량을 잘못 입력했어요.
상세보기 성과 카드에서 다시 ✏ 입력 버튼을 눌러 수정할 수 있습니다. 수정 이력이 자동 기록됩니다.
정산 탭에 캠페인이 보이지 않아요.
정산 탭은 기본적으로 성과입력완료 상태인 캠페인만 표시합니다. 해당 캠페인이 성과입력완료 상태인지 확인하고, 상단 범위 필터가 '전체 캠페인'으로 설정되어 있는지도 확인해 보세요. 또한 연도·월 필터가 해당 캠페인의 발송일과 맞는지 확인하세요.
같은 조건으로 캠페인을 여러 개 등록해야 해요.
기존 캠페인 상세보기에서 복사 등록 버튼을 이용하면 모든 정보가 자동으로 채워진 등록 폼이 열립니다. 발송일시만 새로 입력하면 됩니다.
광고비 지급요청서 다운로드 시 퍼미션콜 데이터는 어디서 확인하나요?
퍼미션콜 항목은 디앤유와 OHC 시트 2개로 분리 생성됩니다. 디앤유 시트에는 DNU 비용, OHC 시트에는 OHC 비용이 표기됩니다.
DA 캠페인에 발송문구를 입력하는 란이 없어요.
DA 캠페인은 문자 발송이 아닌 배너·동영상 소재를 사용하므로 발송문구 입력란이 없습니다. 대신 등록 시 소재 이미지를 첨부하세요.
LMS 등록 시 시간 00:00으로 설정이 안 됩니다.
LMS / MMS / push / 카톡msg 상품은 발송 시간이 필수이기 때문에 00:00 설정이 차단되어 있습니다. 실제 발송 예정 시간을 정확히 입력해 주세요.
캠페인을 삭제하고 싶어요.
캠페인 상세보기 우측 상단 삭제 버튼을 통해 삭제할 수 있습니다. 단, 삭제는 되돌릴 수 없으므로 신중하게 진행하세요. 삭제 권한이 없다면 관리자에게 문의하세요.
캠페인 대시보드 실무자 사용 설명서 · 문의: 광고3본부 황순근책임
