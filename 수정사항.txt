핵심 문제점 (미완성 영역)

  1. 백엔드/DB 없음 — 가장 치명적
  - 모든 데이터가 const DATA = [...], const USERS = [...]로 JS 안에 하드코딩
  - 페이지 새로고침하면 추가/수정한 내용 전부 사라짐
  - 코드 곳곳에 // TODO(DB연동): API 교체 포인트 주석이 10개 이상

  2. 보안 취약
  - 비밀번호가 JS 소스에 평문(pw:'1234')으로 노출
  - 서버 인증 없이 sessionStorage만으로 로그인 상태 판단

  3. 미완성 기능들 (수정사항.txt 기준)
  - 캠페인 등록 드롭박스 표기 (브랜드명_회사명) 미반영
  - 브랜드 등록 시 카테고리 연동 미구현
  - admin 계정의 "나의 캠페인" 섹션 hidden 처리 미구현
  - 월별 지급내역 (nav-payment) 메뉴 주석 처리된 채로 방치
  - LMS/MMS 수수료 단가 구분 데이터 정리 미완
